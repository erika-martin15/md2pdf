# Md2PDF

Convertidor de Markdown a PDF con un diseño moderno y estético.

Este proyecto permite transformar archivos de texto con formato Markdown en documentos PDF profesionales manteniendo una estética legible y cuidada.

## Características

- Editor en tiempo real con vista previa instantánea.
- Exportación directa a PDF con formato A4.
- Diseño accesible y responsivo.
- Opción para personalizar el título del documento final.

## Tecnologías utilizadas

- HTML5 y CSS3 para la estructura y el diseño visual.
- JavaScript para la lógica de procesamiento y exportación.
- Marked.js como motor de procesamiento de Markdown.
- Html2pdf.js para la generación y descarga de archivos PDF.

## Instrucciones de uso

1. Introduzca el contenido en formato Markdown en el panel de edición de la izquierda.
2. Especifique el título deseado para el documento en el campo correspondiente.
3. Revise la vista previa en el panel derecho para asegurar que el formato es correcto.
4. Utilice el botón de exportación para generar y descargar el archivo PDF.
